# 图片轮播视频生成器

> 把图片素材自动生成带推进放大转场的轮播视频，支持背景音乐和尾页拼接。

---

## 功能

- **素材库** — 不限数量添加 JPG/PNG/WEBP/BMP 图片，跨会话记忆
- **随机不重复** — 每个视频图片顺序随机打乱，多个视频保证排列不同
- **推进放大转场** — 类剪映「推进」效果，smoothstep 缓动 + 叠化混合，图片全程静止不抖
- **背景音乐** — MP3/WAV/AAC/FLAC，自动循环截断匹配视频时长，与尾页原声混合
- **尾页视频** — 拼接任意格式尾页视频，自动缩放对齐分辨率
- **画质可调** — 极好 / 高清 / 标准 / 省空间 / 极省 五档，CRF 16~33
- **参数自由** — 图片数、视频时长、生成个数、帧率、转场平滑度、分辨率

## 快速开始

### 1. 安装依赖

```bash
pip install customtkinter opencv-python Pillow numpy
```

### 2. 安装 FFmpeg（可选但推荐）

BGM 合并、尾页拼接、H.264 编码都需要 FFmpeg。

- [下载 FFmpeg](https://ffmpeg.org/download.html)
- 解压后将 `bin` 目录加入系统 PATH

### 3. 运行

```bash
python main.py
```

或双击 `启动.bat`。

## 使用流程

1. 点击「添加图片」导入素材库
2. 左侧面板调参：每视频几张图、多长时间、生成几个
3. 可选：选背景音乐、选尾页视频
4. 点击「生成视频」
5. 完成后自动弹出输出目录

## 依赖

| 库 | 用途 |
|---|---|
| `customtkinter` | 界面框架 |
| `opencv-python` | 视频帧渲染、转场特效 |
| `Pillow` | 图片处理、缩略图 |
| `numpy` | 帧数据运算 |
| `ffmpeg`（外部） | 音视频编码、拼接、混音 |

## 输出

- 格式：MP4 / H.264
- 分辨率：1080×1920（默认，可选横屏/方屏）
- 命名：`slideshow_001_时间戳.mp4`
- 目录：`./output/`

## 参数说明

| 参数 | 默认值 | 范围 |
|---|---|---|
| 每视频图片数 | 8 | 2~30 |
| 视频时长 | 10 秒 | 3~60 秒 |
| 视频个数 | 3 | 1~50 |
| 帧率 | 30 fps | 15~60 |
| 转场平滑度 | 15 帧 | 5~40 |
| 分辨率 | 1080×1920 | 竖/横/方屏 |
| 画质 | 标准 (CRF24) | 五档 |
